# Purchasing LMS — Residential Purchasing Course

## Tech Stack
- Next.js 14 (App Router) + Tailwind CSS
- Prisma + PostgreSQL (Neon)
- NextAuth v4
- Radix UI, TanStack Query, Chart.js
- PDF-lib for certificate generation
- Source on GitHub (Scoremill/purchasing-lms)

## What This Is
A learning management system for residential construction purchasing management training. 12-module course cloned from HowToBuildAHome with purchasing curriculum. Course platform with assessments, progress tracking, and certificate generation.
